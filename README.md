# 班级积分通 (ClassPoints)

一个专门为小学教师设计的开源 Android 学生综合积分管理应用。它可以帮助老师轻松量化考核班级学生的日常表现，支持自定义细则扣分、一键排名、数据可视化分析以及标准 CSV 报表导出。

## 🌟 主要功能

- **多维度评分管理**：严格遵循班级量化考核细则（课堂、作业、纪律、两操、劳动、卫生、安全、个人素质）。
- **自动化计分逻辑**：
  - **本周总分**：基于 100 分基础分，实时统计本周得分。
  - **累积总分**：自动结转往周积分。
  - **周自动重置**：每周一启动时自动将本周积分结算至累积总分并重置周基础分。
- **强大的排序与查询**：
  - 支持按 **学号**、**本周总分**、**累积总分** 三档循环切换排序。
  - 排序状态在应用退出重启后自动保持。
- **数据可视化统计**：
  - **饼状图**：直观展示学生各考核项的得分/扣分占比及具体分值。
  - **折线图**：展示学生本周每日积分的波动趋势（仅展示至当前日期）。
- **报表导出**：支持一键导出符合标准模板的 `.csv` 文件，方便老师上交或打印。
- **极致视觉体验**：适配 Dark/Light 模式，优化系统通知栏显示，确保在大气紫色主题下的完美观感。

## 🛠️ 技术栈

- **开发语言**：Java
- **数据库**：Android Room (持久化学生信息与计分记录)
- **图表库**：MPAndroidChart (实现折线图与饼状图)
- **CSV 解析**：OpenCSV
- **UI 框架**：Material Design 3 (Material Components)
- **视图绑定**：ViewBinding

## 🚀 快速开始

### 编译环境
- Android Studio Iguana 或更高版本
- JDK 17
- Android SDK 35 (Target SDK)

### 使用指南
1. **初始化**：初次使用点击“初始化班级”按钮，输入班级总人数，系统将自动生成学号和默认姓名。
2. **计分**：点击学生卡片右侧的 `+` 按钮，选择大类及具体考核细则即可完成即时计分。
3. **查看详情**：长按学生条目，即可进入详情页面查看多维度数据统计图表。
4. **排序**：多次点击标题下方的排序按钮，可在不同排名逻辑间切换。
5. **导出**：点击“导出CSV”将数据保存至手机外部存储空间。
